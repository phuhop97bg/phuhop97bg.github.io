Pairing System With Facial Images

Introduction

The pairing system is a recommendation system for matching partners based on user input data. Current pairing systems often require users to enter their personal information and desired characteristics of their ideal partners. However, these systems often do not utilize information in the user's photos. Therefore, this project explores the integration of image processing techniques and artificial intelligence algorithms to extract certain facial features and find a suitable partner.
